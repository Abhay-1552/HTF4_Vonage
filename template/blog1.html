Hi, I’m Abhay

and here’s my Vonage API Developer story

BEGIN

I run a blog about my adventures with Chicken — my dog!

I love photographing and writing about my endeavours and funny mishaps with Chicken. It’s my passion, but it’s also a lot of time and effort, and I am always looking for ways to better support myself financially.

Recently, I stumbled upon a concept known as Vonage. It presents a novel approach to integrate sms, voice, video and two-factor authentication into your apps with Vonage communication APIs. Build complex conversational flows with a user friendly drag-and-drop interface in Vonage AI Studio.

I thought it would be cool to experiment with Vonage, which makes it possible for readers to learn different aspects of API Communication, and running and scaling for modern applications as they spend time on my blog.

All I had to do to enable Vonage Authentication on my project was add the API Key and Secret Key with my API request code which is inside the api_request.js file.

Try it for yourself :) Drag the <script> tag below into the <head> tag on the right. 

<link src="api_request.js">
    <html lang="en">
        <head>
            

        </head>
        <body>
            <script>
            </script>
        </body>
    </html>

Now that I've included this single line of code, readers of my website who use a Vonage API Developer can access the API response. This <link> tag with Javascrpit file is all you need to have a API Access!
